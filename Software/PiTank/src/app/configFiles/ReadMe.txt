These config files should be copied into following directories

	appweb			:	copy this directory under /etc
	
	init.d/appweb	:	copy "appweb" file under /etc/init.d/
						give this script execute permission for all
						"sudo chmod 0755 appweb"
						
	www/appweb		: 	copy "appweb" directory under /var/www/
